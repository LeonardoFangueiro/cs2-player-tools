# CS2 Player Tools

## Project Overview
Windows GUI application with tools for CS2 (Counter-Strike 2) players. Categories include connectivity/network diagnostics, VPN optimization, and more.

## Knowledge Base
Comprehensive research docs are in `docs/kb/`:
- `01-valve-infrastructure.md` — Valve SDR, datacenters, connection flow
- `02-cs2-network-protocol.md` — CS2 protocol, sub-tick, ConVars
- `03-network-diagnostics.md` — Diagnostic tools, common problems
- `04-windows-network-optimizations.md` — Windows network tuning
- `05-wireguard-vpn-gaming.md` — WireGuard VPN for gaming
- `06-app-feature-roadmap.md` — Feature roadmap and tech stack options
