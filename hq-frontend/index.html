Installation Notes

1. Download the NSIS (.exe) or MSI installer

2. Run as Administrator (required for network optimizations)

3. WebView2 runtime is included automatically

4. WireGuard binaries are bundled — no separate install needed

Features:

• Network Diagnostics (ping, traceroute, DNS, buffer bloat, MTU)

• Windows Optimizer (7 tweaks with safety levels)

• Smart VPN (auto SSH deploy WireGuard to VPS)

• Server Picker (region lock via firewall)

• CS2 Config Helper (autoexec.cfg optimization)

• System Tray + CS2 Auto-Detection
